# 实验 1 Android开发基础

## 实验环境

- 操作系统：Windows 11
- 软件工具：
  - Android Studio Koala 2024
  - Git 版本 2.47.1

## 实验步骤

### 1. 安装 Android Studio

1. **下载 Android Studio**
   - 官方下载地址：
     - https://developer.android.com/studio
     - https://developer.android.google.cn/studio/
2. **安装注意事项**
   - Android Studio 版本越高，对电脑硬件系统要求越高。如果硬件达不到要求，可以选择更早的版本。
3. **完成安装**
   - 安装完成后，新建第一个项目时，会同步项目的依赖文件和支持库。

### 2. 注册 GitHub 账号并安装 Git

1. **注册 GitHub 账号**
   - [GitHub 官方地址](https://github.com)
2. **下载并安装 Git**
   - 下载地址：https://git-scm.com/
   - 安装版本：2.47.1

### 3. 创建 Android 工程并同步至 GitHub

1. **创建工程**
   - 使用 Android Studio 创建一个新的 Android 项目。
2. **同步至 GitHub**
   - 参考方式：命令行方式（推荐）。
   - 参考博客：http://blog.csdn.net/fjnu_se/article/details/66472625
3. **添加忽略文件**
   - 使用 GitHub 提供的 Android .gitignore 文件：https://github.com/github/gitignore/blob/master/Android.gitignore

## 实验总结

通过本次实验，我们完成了以下任务：

- 安装并配置了 Android Studio。
- 注册了 GitHub 账号并安装了 Git。
- 成功创建了第一个 Android 工程，并将其同步至 GitHub。

通过实验，我们熟悉了开发环境的搭建流程，为后续的 Android 开发打下了坚实的基础。
